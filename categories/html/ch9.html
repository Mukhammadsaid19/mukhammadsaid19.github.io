HTML formatlash elementlari

Formatlash elementlari maxsus turdagi matnlarni ko'rsatish uchun mo'ljallangan:

  • <b> - Qalin matn
  • <strong> - Muhim matn
  • <i> - Kursiv matn
  • <em> - Ta'kidlangan matn
  • <mark> - Belgilangan matn
  • <small> - Kichikroq matn
  • <del> - O'chirilgan matn
  • <ins> - Kiritilgan matn
  • <sub> - Subscript matni
  • <sup> - Superscript matni

HTML <b> va <strong> Elementlar

HTML <b> elementi hech qanday qo'shimcha ahamiyatga ega bo'lmagan holda qalin matnni belgilaydi.

<b> Ushbu matn qalin </b>

Ushbu matn qalin

HTML <strong> elementi matnning katta ahamiyatga egaligini belgilaydi. Ichidagi tarkib, odatda, qalin harflar bilan ko'rsatiladi.

<strong> Ushbu matn muhim! </strong>

Ushbu matn muhim!

HTML <i> va <em> elementlari

HTML <i> elementi matnning bir qismini muqobil ovoz yoki kayfiyatda belgilaydi. Ichidagi tarkib, odatda, kursiv bilan ko'rsatiladi.

Maslahat: <i> tegi ko'pincha texnik atama, boshqa tildan olingan ibora, fikr, kema nomi va boshqalarni ko'rsatish uchun ishlatiladi.

<i> Ushbu matn kursiv </i>

Ushbu matn kursiv

HTML <em> elementi ta'kidlangan matnni belgilaydi. Ichidagi tarkib, odatda, kursiv bilan ko'rsatiladi.

Maslahat: Ekranni o'quvchi og'zaki stress yordamida <em> dagi so'zlarni diqqat bilan talaffuz qiladi.

<em> Ushbu matn ta'kidlangan </em>

Ushbu matn ta'kidlangan

HTML <small> elementi

HTML <small> elementi kichikroq matnni belgilaydi:

<small> Bu kichikroq matn. </small>

Bu kichikroq matn.

HTML <mark> elementi

HTML <mark> elementi belgilanishi yoki ajratilishi kerak bo'lgan matnni belgilaydi:

<p> Bugun <mark>sut</mark> sotib olishni unutmang. </p>

Bugun sut sotib olishni unutmang.

HTML <del> elementi

HTML <del> elementi hujjatdan o'chirilgan matnni belgilaydi. Brauzerlar, odatda, o'chirilgan matnlar qatoriga gorizontal chiziq bilan ko'rsatadi:

<p> Mening eng yaxshi ko'rgan rangim <del>ko'k</del> qizil. </p>

Mening eng yaxshi ko'rgan rangim ko'k qizil.

HTML <ins> elementi

HTML <ins> elementi hujjatga kiritilgan matnni belgilaydi. Brauzerlar, odatda, kiritilgan matn ostiga chizishadi:

<p> Mening eng yaxshi ko'rgan rangim <del>ko'k</del> <ins>qizil</ins>. </p>

Mening eng yaxshi ko'rgan rangim ko'k qizil.

HTML <sub> elementi

HTML <sub> elementi pastki yozuv matnini belgilaydi. Subscript matni odatdagi satrdan yarim belgi ostida ko'rinadi va ba'zida kichikroq shriftda ko'rsatiladi. Subscript matni H2O kabi kimyoviy formulalar uchun ishlatilishi mumkin:

<p> Bu <sub>pastda yozilgan</sub> matn. </p>

Bu pastda yozilgan matn.

HTML <sup> elementi

HTML <sup> elementi yuqorida yozilgan matnni belgilaydi. Superscript matni odatdagi satrdan yarim belgidan yuqori bo'lib ko'rinadi va ba'zida kichikroq shriftda ko'rsatiladi. Superscript matni shu kabi izohlar uchun ishlatilishi mumkin: WWW [1]:

<p> Bu <sup>yuqorida yozilgan</sup> matn.</p>

Bu yuqorida yozilgan matn.
