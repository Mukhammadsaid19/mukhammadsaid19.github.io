HTML jadvalini belgilang

<table> yorlig'i HTML jadvalini belgilaydi.

Har bir jadval satri <tr> yorlig'i bilan aniqlanadi. Har bir jadval sarlavhasi <th> yorlig'i bilan aniqlanadi. Har bir jadval ma'lumotlari / katakchasi <td> yorlig'i bilan aniqlanadi.

Odatda, <th> elementlaridagi matn qalin va o'rtada joylashgan.

Odatda, <td> elementlaridagi matn muntazam va chapga tekislangan.

Misol

Oddiy HTML jadval:

<table style="width:100%">
  <tr>
    <th>Firstname</th>
    <th>Lastname</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Jill</td>
    <td>Smith</td>
    <td>50</td>
  </tr>
  <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>94</td>
  </tr>
</table>

HTML jadvali - Chegarani qo'shish

Jadvalga chegara qo'shish uchun CSS border xususiyatidan foydalaning:

Misol

table, th, td {
  border: 1px solid black;
}

Ham jadval, ham jadval kataklari uchun chegaralarni belgilashni unutmang.

HTML jadvali - Yig'ilgan chegaralar

Chegaralarning bitta chegaraga tushishiga ruxsat berish uchun CSS border-collapse xususiyatini qo'shing:

Misol

table, th, td {
  border: 1px solid black;
  border-collapse: collapse;
}

HTML jadvali - Katakchani to'ldirilishini qo'shish

Katakcha to'ldirilishi katak tarkibi va uning chegaralari orasidagi bo'shliqni aniqlaydi.

Agar siz to'ldirishni ko'rsatmasangiz, jadval katakchalari to'ldirilmasdan ko'rsatiladi.

To'ldirishni o'rnatish uchun CSS padding xususiyatidan foydalaning:

Misol

th, td {
  padding: 15px;
}

HTML jadvali - chapga tekislangan sarlavhalar

Odatda, jadval sarlavhalari qalin va markazlashtirilgan.

Jadval sarlavhalarini chapga tekislash uchun CSS text-align xususiyatidan foydalaning:

Misol

th {
  text-align: left;
}

HTML jadvali - chegara oralig'ini qo'shish

Chegaralar oralig'i kataklar orasidagi bo'shliqni aniqlaydi.

Jadval uchun chegara oralig'ini o'rnatish uchun border-spacing CSS xususiyatidan foydalaning:

Misol

table {
  border-spacing: 5px;
}

Eslatma: Agar jadval chegaralari border-collapse bo'lsa, chegara oralig'i hech qanday ta'sir qilmaydi.

HTML jadvali - ko'plab ustunlarni qamrab oluvchi katak

Katakni bitta ustundan ko'proq qilish uchun colspan atributidan foydalaning:

Misol

<table style="width:100%">
  <tr>
    <th>Name</th>
    <th colspan="2">Telephone</th>
  </tr>
  <tr>
    <td>Bill Gates</td>
    <td>55577854</td>
    <td>55577855</td>
  </tr>
</table>

HTML jadvali - sarlavha qo'shish

Jadvalga sarlavha qo'shish uchun <caption> tegidan foydalaning:

<table style="width:100%">
  <caption>Monthly savings</caption>
  <tr>

    <th>Savings</th>
  </tr>

  </tr>

  </tr>

Eslatma: <caption> yorlig'i <table> tegidan so'ng darhol kiritilishi kerak.

Bitta jadval uchun maxsus stillar

Muayyan jadval uchun maxsus uslubni aniqlash uchun jadvalga id atributini qo'shing:

<table id="t01">
  <tr>
    <th>Firstname</th>
    <th>Lastname</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>94</td>
  </tr>
</table>
Firstname Lastname Age
Eve Jackson 94

Endi ushbu jadval uchun maxsus uslubni belgilashingiz mumkin:

#t01 {
  width: 100%;
  background-color: #f1f1c1;
}

Va ko'proq stillarni qo'shing:

#t01 tr:nth-child(even) {
  background-color: #eee;
}
#t01 tr:nth-child(odd) {
  background-color: #fff;
}
#t01 th {
  color: white;
  background-color: black;
}
