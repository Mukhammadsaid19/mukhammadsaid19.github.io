HTML ranglari

HTML ranglar oldindan belgilangan rang nomlari yoki RGB, HEX, HSL, RGBA yoki HSLA qiymatlari bilan belgilanadi.

Rang nomlari

Orange

DodgerBlue

MediumSeaGreen

Gray

SlateBlue

Violet

LightGray

HTML 140 standart rang nomlarini qo'llab-quvvatlaydi.

Rang qiymatlari

HTMLda ranglarni RGB qiymatlari, HEX qiymatlari, HSL qiymatlari, RGBA qiymatlari va HSLA qiymatlari yordamida ham ko'rsatish mumkin.

Quyidagi uchta <div> elementlari fon rangini RGB, HEX va HSL qiymatlari bilan o'rnatgan:

rgb(255, 99, 71)
#ff6347
hsl(9, 100%, 64%)

Quyidagi ikkita <div> elementi fon rangini RGBA va HSLA qiymatlari bilan o'rnatilgan, bu rangga alpha kanalini qo'shadi (bu yerda bizda 50% shaffoflik mavjud):

rgba(255, 99, 71, 0.5)
hsla(9, 100%, 64%, 0.5)

Misol

<h1 style = "background-color: rgb (255, 99, 71);"> ... </h1>
<h1 style = "background-color: # ff6347;"> ... </h1>
<h1 style = "background-color: hsl (9, 100%, 64%);"> ... </h1>

<h1 style = "background-color: rgba (255, 99, 71, 0.5);"> ... </h1>
<h1 style = "background-color: hsla (9, 100%, 64%, 0.5);"> ... </h1>

Rang qiymatlari haqida ko'proq bilib oling

RGB, HEX va HSL haqida keyingi boblarda bilib olasiz.
