HTML Rasmlari sintaksisi

HTML <img> yorlig'i veb-sahifaga rasm joylashtirish uchun ishlatiladi.

Rasmlar veb-sahifaga texnik jihatdan kiritilmagan; rasmlar veb-sahifalarga bog'langan. <img> yorlig'i havola qilingan rasm uchun bo'sh joy yaratadi.

<img> tegi bo'sh, u faqat atributlarni o'z ichiga oladi va yopilish yorlig'iga ega emas.

<img> tegida ikkita zarur atribut mavjud:

src - rasmga olib boradigan yo'lni belgilaydi alt - rasm uchun muqobil matnni belgilaydi

Sintaksis

<img src="url" alt="yozuv">

src atributi

Majburiy src atributi rasmga yo'lni (URL) belgilaydi.

Izoh: veb-sahifa yuklangan paytda brauzer veb-serverdan rasmni oladi va uni sahifaga qo'shadi. Shuning uchun, rasm aslida veb-sahifaga nisbatan bir xil joyda turishiga ishonch hosil qiling, aks holda sizning tashrif buyuruvchilaringiz buzilgan havola belgisini olishadi. Agar brauzer rasmni topa olmasa, buzilgan havola belgisi va pastki matn ko'rsatiladi.

<img src="img_chania.jpg" alt="Chaniyadagi Gullar">

alt atributi

Kerakli alt atributi rasm uchun muqobil matnni taqdim etadi, agar foydalanuvchi biron sababga ko'ra uni ko'rib chiqa olmasa (ulanish sustligi, src atributidagi xato yoki foydalanuvchi ekran o'quvchisidan foydalansa).

alt atributining qiymati tasvirni tavsiflashi kerak:

<img src="img_chania.jpg" alt="Chaniyadagi Gullar">

Agar brauzer rasm topa olmasa, u alt atributining qiymatini aks ettiradi:

Rasm o'lchami - width va height

Rasmning kengligi va balandligini ko'rsatish uchun siz stil atributidan foydalanishingiz mumkin.

<img src="img_girl.jpg" alt="Ko'ylagi kiygan qiz" style="width:500px;height:600px;">

Shu bilan bir qatorda, width (kenglik) va height (balandlik) atributlaridan ham foydalanishingiz mumkin:

<img src = "img_girl.jpg" alt = "Ko'ylagi kiygan qiz" width = "500" height = "600">

width va height atributlari har doim rasm kengligi va balandligini piksel bilan belgilaydi.

Izoh: Har doim rasmning kengligi va balandligini ko'rsating. Agar kenglik va balandlik ko'rsatilmagan bo'lsa, rasm yuklanganda, veb-sahifa miltillashi mumkin.

Kenglik va balandlikmi yoki stillarmi?

width, height va style atributlari barchasi HTMLda amal qiladi.

Biroq, style atributidan foydalanishni taklif qilamiz. Bu uslublar jadvallarini rasm hajmini o'zgartirishga to'sqinlik qiladi:

<!DOCTYPE html>
<html>
<head>
<style>
img {
  width: 100%;
}
</style>
</head>
<body>

<img src="html5.gif" alt="HTML5 Icon" width="128" height="128">

<img src="html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">

</body>
</html>

Boshqa papkadagi rasmlar

Agar rasmlaringiz pastki papkada bo'lsa, src atributiga papka nomini kiritishingiz kerak:

<img src="/images/html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">

Boshqa server/veb-saytdagi rasmlar

Ba'zi veb-saytlar boshqa serverdagi rasmga ishora qiladi.

Boshqa serverdagi rasmga ishora qilish uchun src atributida mutlaq (to'liq) URL manzilini ko'rsatishingiz kerak:

<img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools.com">

Tashqi rasmlarga oid eslatmalar: Tashqi rasmlar mualliflik huquqiga ega bo'lishi mumkin. Agar siz undan foydalanishga ruxsat olmasangiz, mualliflik huquqi to'g'risidagi qonunlarni buzgan bo'lishingiz mumkin. Bundan tashqari, siz tashqi tasvirlarni boshqarolmaysiz; uni to'satdan olib tashlash yoki o'zgartirish mumkin.

Rasm Joylashuvi

Joylashishi matnning o'ng yoki chap tomoniga o'tishi uchun CSS float xususiyatidan foydalaning:

Misol

<p> <img src = "smiley.gif" alt = "Smiley face" style = "float: right; width: 42px; height: 42px;">
Rasm matnning o'ng tomonida suzadi.</p>

<p> <img src = "smiley.gif" alt = "Smiley face" style = "float: left; width: 42px; height: 42px;">
Rasm matnning chap tomonida suzadi.</p>

Bo'limning qisqacha mazmuni

  • Rasmni aniqlash uchun HTML <img> elementidan foydalaning
  • Rasmning URL manzilini aniqlash uchun HTML src atributidan foydalaning
  • HTML alt atributidan foydalanib, rasm uchun muqobil matnni aniqlang, agar uni ko'rsatish mumkin bo'lmasa
  • Rasm hajmini aniqlash uchun HTML width va height atributlaridan yoki CSS widthva height xususiyatlaridan foydalaning
  • CSS float xususiyatidan foydalanib, rasmni chapga yoki o'ngga joylashitiring
