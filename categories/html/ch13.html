• 13. HTML havolalari

HTML havolalari

Havolalar deyarli barcha veb-sahifalarda joylashgan. Havolalar foydalanuvchilarga sahifadan sahifaga o'tish imkonini beradi.

HTML havolalari - giperhavolalar

HTML havolalari bu – giperhavolalardir.

Siz havolani bosishingiz va boshqa hujjatga o'tishingiz mumkin.

Sichqonchani havola orqali siljitganda, sichqoncha o'qi kichik qo'lga aylanadi.

Izoh: Havola matn bo'lishi shart emas. Havola rasm yoki boshqa HTML element bo'lishi ham mumkin!

HTML havolalari - sintaksis

HTML <a> yorlig'i ko'prikni belgilaydi. U quyidagi sintaksisga ega:

<a href="url">havola matni</a>

<a> elementining eng muhim atributi bu – href atributidir, u havolaning borishini ko'rsatadi.

Havola matni o'quvchiga ko'rinadigan qismdir.

Havola matnini bosish orqali o'quvchi ko'rsatilgan URL manziliga yuboriladi.

Misol

Ushbu misol dr.rtm.uz/coding-club saytida havolani qanday yaratishni ko'rsatadi:

<a href="https://dr.rtm.uz/coding-club/">Coding Club ga marhamat!</a>

Odatiy bo'lib, barcha brauzerlarda havolalar quyidagicha ko'rinadi:

  • Ko'rinmagan havola chizilgan va ko'k rangda
  • Tashrif buyurilgan havola chizilgan va binafsha rangda
  • Faol havola chizilgan va qizil rangda

HTML havolalari - target atributi

Odatiy bo'lib, bog'langan sahifa joriy brauzer oynasida ko'rsatiladi. Buni o'zgartirish uchun siz havola uchun boshqa maqsadni belgilashingiz kerak.

target atributi bog'langan hujjatni qaerda ochishni belgilaydi.

target atribut quyidagi qiymatlardan biriga ega bo'lishi mumkin:

  • _self - Standart. Hujjatni xuddi shu oynada / yorliqda bosilganda ochadi
  • _blank - Hujjatni yangi oynada yoki yorliqda ochadi
  • _parent - Hujjatni ota-kadrda ochadi
  • _top - Hujjatni oynaning to'liq qismida ochadi

Misol

Bog'langan hujjatni yangi brauzer oynasi yoki yorlig'ida ochish uchun target = "_ blank" dan foydalaning:

<a href="https://dr.rtm.uz/coding-club/" target="_blank">Coding Club ga marhamat!</a>

Mutlaq URL va nisbiy URL manzillari

Yuqoridagi ikkala misol href atributida mutlaq URL (to'liq veb-manzil) dan foydalanmoqda.

Mahalliy havola (xuddi shu veb-saytdagi sahifaga havola) nisbiy URL bilan ko'rsatilgan ("https: // www" qismisiz):

<h2>Mutlaq URL manzillari</h2>
<p><a href="https://dr.rtm.uz/coding-club/">Coding Club</a></p>
<p><a href="https://www.google.com/">Google</a></p>

<h2>Nisbiy URL manzillari</ h2>
<p><a href="html_images.asp">HTML rasmlari</a></p>
<p><a href="/css/default.asp">CSS qo'llanmasi</a></p>

HTML havolalari - Tasvirni Havola sifatida foydalaning

Tasvirni havola sifatida ishlatish uchun faqat <a> yorlig'i ichiga <img> yorlig'ini qo'ying:

Misol

<a href="default.asp">
<img src="smiley.gif" alt="HTML tutorial" style="width:42px;height:42px;">
</a>

Elektron pochta manziliga havola

Foydalanuvchining elektron pochta dasturini ochadigan havolani yaratish uchun (yangi elektron pochta xabarlarini yuborishlariga ruxsat berish uchun) href atributi ichida mailto: dan foydalaning:

<a href="mailto:someone@example.com">Elektron pochta xabarlarini yuboring</a>

Havola sifatida tugma

HTML tugmachasini havola sifatida ishlatish uchun siz ba'zi JavaScript kodlarini qo'shishingiz kerak.

JavaScript sizga ba'zi bir voqealarda nima sodir bo'lishini, masalan, tugmachani bosishni belgilash imkonini beradi:

<button onclick = "document.location = 'default.asp'">HTML qo'llanmasi </button>

Havolalar sarlavhasi

title atributi element haqida qo'shimcha ma'lumotlarni aniqlaydi. Ma'lumot ko'pincha sichqoncha element ustida harakatlanayotganda ko'rsatmalar matni sifatida ko'rsatiladi.

<a href="https://dr.rtm.uz/coding-club/" title="Coding Club ga o'tish" >Coding Club ga marhamat!</a>

Bo'limning qisqacha mazmuni

  • Havolani qo'shish uchun <a> elementidan foydalaning
  • Havola manzilini belgilash uchun href atributidan foydalaning
  • Bog'langan hujjatni qayerda ochishni aniqlash uchun target atributidan foydalaning
  • Tasvirni havola sifatida ishlatish uchun <img> elementidan foydalaning (ichida <a>)
  • Foydalanuvchining elektron pochta dasturini ochadigan havola yaratish uchun href atributi ichidagi mailto: sxemasidan foydalaning
