CSS nima?

Cascading Style Sheets (CSS) veb-sahifaning joylashishini formatlash uchun ishlatiladi.

CSS yordamida siz rang, shrift, matn hajmi, elementlar orasidagi masofa, elementlarning joylashuvi va joylashishi, qanday fon rasmlari yoki ranglarining ishlatilishi, turli qurilmalar uchun har xil displeylar va ekran o'lchamlari kabilarni formatlashingiz mumkin!

Maslahat: Kaskad so'zi shuni anglatadiki, ota-ona elementiga qo'llaniladigan uslub ota-ona tarkibidagi barcha bolalar elementlariga ham tegishlidir. Shunday qilib, agar siz asosiy matn rangini "ko'k" ga qo'ysangiz, tanadagi barcha sarlavhalar, xatboshilar va boshqa matn elementlari ham bir xil rangga ega bo'ladi (agar siz boshqa narsani ko'rsatmasangiz)!

CSS-dan foydalanish

CSSni HTML hujjatlariga 3 usulda qo'shish mumkin:

  • Inline - HTML elementlari ichida stil atributidan foydalanish
  • Internal <head> qismida <style> elementi yordamida
  • External - tashqi CSS fayliga ulanish uchun <link> elementi yordamida

Inline CSS

Inline CSS yagona uslubdagi HTML elementiga noyob uslubni qo'llash uchun ishlatiladi.

Ichki CSS HTML elementining <style> atributidan foydalanadi.

Quyidagi misol

elementi matn rangini ko'k rangga,

elementi matn rangini qizil rangga o'rnatadi:

Misol

<h1 style = "color: blue;"> Moviy sarlavha </h1>

<p style = "color: red;"> Qizil paragraf. </p>

Moviy sarlavha

qizil paragraf.

Ichki CSS

Ichki CSS bitta HTML-sahifa uchun uslubni aniqlash uchun ishlatiladi.

Ichki CSS HTML-sahifaning <head> qismida, <style> elementi ichida aniqlanadi.

Quyidagi misol BARCHA <h1> elementlari matn rangini (o'sha sahifada) ko'k rangga, BARCHA <p> elementlari matn rangini qizil rangga o'rnatadi. Bundan tashqari, sahifa "powderblue" fon rangi bilan namoyish etiladi:

<!DOCTYPE html>
<html>
<bosh>
<style>
tanasi {background-color: powderblue;}
h1 {color: blue;}
p {color: red;}
</style>
</head>
<body>

<h1> Bu sarlavha </ h1>
<p> Bu xatboshi. </p>

</body>
</html>

Tashqi CSS

Tashqi uslublar jadvali ko'plab HTML-sahifalar uchun uslubni aniqlash uchun ishlatiladi.

Tashqi uslublar jadvalidan foydalanish uchun har bir HTML-sahifaning <head> bo'limiga havolani qo'shing:

<!DOCTYPE html>
<html>
<bosh>
   <link rel = "stylesheet" href = "styles.css">
</head>
<body>

<h1> Bu sarlavha </ h1>
<p> Bu xatboshi. </p>

</body>
</html>

Tashqi uslublar jadvalini har qanday matn muharririda yozish mumkin. Faylda hech qanday HTML kod bo'lmasligi kerak va .css kengaytmasi bilan saqlanishi lozim.

"styles.css" fayli shunga o'xshash bo'lishi mumkin:

"styles.css":

body {
  background-color: powderblue;
}
h1 {
  color: blue;
}
p {
  color: red;
}

Maslahat: Tashqi uslublar jadvalida bitta faylni o'zgartirish bilan, butun veb-sayt ko'rinishini o'zgartirishingiz mumkin!

CSS ranglari, shriftlari va o'lchamlari

Bu yerda biz tez-tez ishlatiladigan ba'zi bir CSS xususiyatlarini namoyish etamiz. Keyinchalik ular haqida ko'proq bilib olasiz.

CSS rang xususiyati foydalaniladigan matn rangini belgilaydi.

CSS shrift-oilaviy xususiyati foydalaniladigan shriftni belgilaydi.

CSS shrift hajmi xususiyati foydalaniladigan matn hajmini belgilaydi.

Misol

CSS rang, shrift turkumi va shrift kattaligi xususiyatlaridan foydalanish:

<!DOCTYPE html>
<html>
<head>
<style>
h1 {
  color: blue;
  font-family: verdana;
  font-size: 300%;
}
p {
  color: red;
  font-family: courier;
  font-size: 160%;
}
</style>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

CSS Border

CSS border xususiyati HTML elementi atrofidagi chegarani belgilaydi.

Izoh: Siz deyarli barcha HTML elementlari uchun chegarani belgilashingiz mumkin.

Misol

p {
  border: 2px solid powderblue;
}

CSS Padding

CSS padding xususiyati matn va hoshiya orasidagi to'ldirish (bo'shliq)ni belgilaydi.

Misol

p {
  border: 2px solid powderblue;
  padding: 30px;
}

CSS margin

CSS margin xususiyati chegaradan tashqaridagi chegara (bo'shliq)ni belgilaydi.

Misol

p {
  border: 2px solid powderblue;
  margin: 50px;
}

Tashqi CSS-ga havola

Tashqi uslublar jadvallariga to'liq URL yoki joriy veb-sahifaga nisbatan yo'l bilan murojaat qilish mumkin.

Misol

Ushbu misol uslublar jadvaliga havola qilish uchun to'liq URL manzilidan foydalanadi:

<link rel="stylesheet" href="https://www.w3schools.com/html/styles.css">

Misol

Ushbu misol joriy veb-saytdagi html papkasida joylashgan uslublar jadvaliga havola:

<link rel="stylesheet" href="/html/styles.css">

Misol

Ushbu misol joriy sahifa bilan bir xil papkada joylashgan uslublar jadvaliga havola:

<link rel="stylesheet" href="styles.css">

Fayl yo'llari haqida ko'proq ma'lumotni HTML fayl yo'llari bo'limida o'qishingiz mumkin.

Bo'limning qisqacha mazmuni

  • Ichki uslublar uchun HTML <style> atributidan foydalaning
  • Ichki CSSni aniqlash uchun HTML <style> elementidan foydalaning
  • Tashqi CSS-faylga murojaat qilish uchun HTML <link> elementidan foydalaning
  • <style> va <link> elementlarini saqlash uchun HTML <head> elementidan foydalaning
  • Matn ranglari uchun CSS color xususiyatidan foydalaning
  • Matn shriftlari uchun CSS font-family xususiyatidan foydalaning
  • Matn o'lchamlari uchun CSS font-size xususiyatidan foydalaning
  • Chegaralar uchun CSS border xususiyatidan foydalaning
  • Chegaradagi bo'sh joy uchun CSS padding xususiyatidan foydalaning
  • Chegaradan tashqarida bo'sh joy uchun CSS margin xususiyatidan foydalaning
