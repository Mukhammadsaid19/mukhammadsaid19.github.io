• 15. HTML Fon Rasmlari

HTML Fon Rasmlari

Fon rasmini deyarli har qanday HTML elementi uchun ko'rsatish mumkin.

HTML elementidagi fon rasmi

HTML elementiga fon rasmini qo'shish uchun HTML style atributi va CSS background-color xususiyatidan foydalaning:

Misol

HTML elementiga fon rasmini qo'shing:

<div style="background-image: url('img_girl.jpg');">

Siz <style> elementida <head> bo'limida fon rasmini belgilashingiz mumkin:

Misol

<style> elementida fon rasmini ko'rsating:

<style>
div {
  background-image: url('img_girl.jpg');
}
</style>

Sahifadagi orqa fon rasmi

Agar siz butun sahifa fon rasmiga ega bo'lishini istasangiz, <body> elementida fon rasmini ko'rsatishingiz kerak:

Misol

Butun sahifa uchun fon rasmining qo'shilishi:

<style>
body {
  background-image: url('img_girl.jpg');
}
</style>

Orqa fonda takrorlash

Agar fon rasmi elementdan kichikroq bo'lsa, rasm elementning oxirigacha gorizontal va vertikal ravishda takrorlanadi:

Misol

<style>
body {
  background-image: url('example_img_girl.jpg');
}
</style>

Fon rasmining takrorlanishiga yo'l qo'ymaslik uchun, background-repeat xususiyatini no-repeat qilib o'rnating.

Misol

<style>
body {
  background-image: url('example_img_girl.jpg');
  background-repeat: no-repeat;
}
</style>

Orqa fon usti

Agar siz fon rasmi butun elementni qamrab olishini istasangiz, siz background-size xususiyatini cover qilib o'rnatishingiz mumkin.

Bundan tashqari, butun element har doim yopiq ekanligiga ishonch hosil qilish uchun, background-attachment xususiyatini fixed: qilib qo'ying:

Shunday qilib, fon rasmi butun elementni qamrab oladi, hech qanday cho'zilmasdan (rasm asl nisbatlarini saqlab qoladi):

<style>
body {
  background-image: url('img_girl.jpg');
  background-repeat: no-repeat;
  background-attachment: fixed;
  background-size: cover;
}
</style>

Orqa fonni cho'zish

Agar siz fon rasmini butun elementga mos kelishini xohlasangiz, siz background-size xususiyatini 100% 100% ga o'rnatishingiz mumkin:

Brauzer oynasining o'lchamini o'zgartirishga urinib ko'ring, shunda rasm cho'zilib ketishini ko'rasiz, lekin har doim butun elementni qamrab olasiz.
