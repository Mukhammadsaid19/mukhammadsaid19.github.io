• 11.1. HTML RGB va RGBA ranglari

HTML RGB va RGBA ranglari

RGB rang qiymati RED, GREEN va BLUE yorug'lik manbalarini aks ettiradi.

RGBA rang qiymati - alfa kanali (xiralik) bilan RGB kengaytmasi.

RGB rang qiymatlari

HTMLda rang quyidagi formuladan foydalangan holda RGB qiymati sifatida ko'rsatilishi mumkin:

rgb (red (qizil), green (yashil), blue (ko'k))

Har bir parametr (qizil, yashil va ko'k) rangning qizg'inligini 0 dan 255 gacha bo'lgan qiymat bilan belgilaydi.

Bu shuni anglatadiki, 256 x 256 x 256 = 16777216 rang bo'lishi mumkin!

Masalan, rgb (255, 0, 0) qizil rangda ko'rsatiladi, chunki qizil eng yuqori qiymatiga (255), qolgan ikkitasi (yashil va ko'k) 0 ga o'rnatiladi.

Yana bir misol, rgb (0, 255, 0) yashil rangda ko'rsatiladi, chunki yashil eng yuqori qiymatiga (255), qolgan ikkitasi (qizil va ko'k) 0 ga o'rnatiladi.

Qora rangni ko'rsatish uchun barcha rang parametrlarini 0 ga qo'ying, quyidagicha: rgb (0, 0, 0).

Oq rangni ko'rsatish uchun barcha rang parametrlarini quyidagicha o'rnating: rgb (255, 255, 255).

Quyidagi RGB qiymatlarini aralashtirib tajriba qiling:

Misol

rgb(255, 0, 0)

rgb(0, 0, 255)

rgb(60, 179, 113)

rgb(238, 130, 238)

rgb(255, 165, 0)

rgb(106, 90, 205)

Kulrang soyalar

Kulrang soyalar ko'pincha uchta parametr uchun teng qiymatlar yordamida aniqlanadi:

Misol

rgb(60, 60, 60)

rgb(100, 100, 100)

rgb(140, 140, 140)

rgb(180, 180, 180)

rgb(200, 200, 200)

rgb(240, 240, 240)

RGBA rang qiymatlari

RGBA rang qiymatlari - RGB rang qiymatlarining Alpha kanali bilan kengaytirilishi. Alpha kanal rangning xiralashishini belgilaydi.

RGBA rang qiymati quyidagicha ko'rsatilgan:

rgba (qizil, yashil, ko'k, alfa)

Alfa parametri 0,0 (to'liq shaffof) va 1,0 (umuman shaffof emas) orasidagi raqam:

rgba(255, 99, 71, 0)

rgba(255, 99, 71, 0.2)

rgba(255, 99, 71, 0.4)

rgba(255, 99, 71, 0.6)

rgba(255, 99, 71, 0.8)

rgba(255, 99, 71, 1)
