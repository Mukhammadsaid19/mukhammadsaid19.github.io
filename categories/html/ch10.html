• 10. HTML izohlari

HTML izohlari

HTML izohlari brauzerda ko'rsatilmaydi, lekin ular HTML manba kodingizni hujjatlashtirishga yordam beradi.

HTML izoh teglari

Quyidagi sintaksis yordamida HTML manbangizga izohlar qo'shishingiz mumkin:

<! - O'z izohlaringizni bu erga yozing ->

E'tibor bering, boshlang'ich yorlig'ida undov belgisi bor (!), lekin oxirgi yorliqda emas.

Muhim: Izohlar brauzer tomonidan ko'rsatilmaydi, lekin ular HTML manba kodingizni hujjatlashtirishga yordam beradi.

Izohlar bilan siz HTML kodingizga bildirishnomalar va eslatmalarni joylashtirishingiz mumkin:

Misol

<! - Bu izoh ->

<p> Bu xatboshi. </p>

<! - Qo'shimcha ma'lumotni bu erga qo'shishni unutmang ->

Bu xatboshi.

Izohlar HTMLni xatolarga tekshirish (otladka qilish) uchun ham juda yaxshi, chunki siz xatolarni qidirish uchun HTML satrlarini birma-bir izohlashingiz mumkin:

<! - Ayni paytda ushbu rasmni namoyish qilmang
<img border = "0" src = "pic_trulli.jpg" alt = "Trulli">
->
