HTML style atributi

HTML elementi uslubini sozlash style atributi bilan amalga oshirilishi mumkin.

HTML style atributi quyidagi sintaksisga ega:

<tegnomi style = "property : value ;">

property CSS xususiyatidir. value CSS qiymatidir.

Ushbu qo'llanmada CSS haqida ko'proq ma'lumotga ega bo'lasiz.

Fon rangi

CSS background-color xususiyati HTML elementi uchun fon rangini belgilaydi.

Misol

Sahifa uchun fon rangini chang-ko'k (powderblue) rangda o'rnating:

<body style = "background-color: powderblue;">

<h1> Bu sarlavha </ h1>
<p> Bu xatboshi. </p>

</body>

Misol

Ikki xil element uchun fon rangini o'rnating:

<body>

<h1 style = "background-color: powderblue;"> Bu sarlavha </h1>
<p style = "background-color: tomato;"> Bu xatboshi. </p>

</body>

Matn rangi

CSS color xususiyati HTML elementi uchun matn rangini belgilaydi:

Misol

<h1 style = "color: blue;"> Bu sarlavha </h1>
<p style = "color: red;"> Bu paragraf. </p>

Bu sarlavha

Bu paragraf.

Shriftlar

CSS font-family atributi HTML elementi uchun ishlatiladigan shriftni belgilaydi:

<h1 style = "font-family: verdana;"> Bu sarlavha </h1>
<p style = "font-family: courier;"> Bu paragraf. </p>

Bu sarlavha

Bu paragraf.

Matn hajmi

CSS font-size xususiyati HTML elementi uchun matn hajmini belgilaydi:

Misol

<h1 style = "font-size: 300%;"> Bu sarlavha </h1>
<p style = "font-size: 160%;"> Bu paragraf. </p>

Bu sarlavha

Bu paragraf.

Matnni joylashuvi

CSS text-align xususiyati HTML elementi uchun gorizontal matn tekisligini belgilaydi:

Misol

Markazlashtirilgan paragraf.
